은하 지도 배경 이미지 안내
--------------------------------
galaxy-map.html은 이 폴더의 galaxy.jpg를 우주 배경으로 사용합니다.
없으면 자체 제작 나선팔로 자동 폴백되므로 비워둬도 동작합니다.

권장: NASA 공개 도메인 나선 은하 이미지를 받아 assets/galaxy.jpg 로 저장
 - images.nasa.gov 에서 "spiral galaxy" / "Milky Way" 검색
 - 예: Pinwheel Galaxy (M101), Andromeda (M31), 또는 은하수 파노라마
 - NASA 이미지는 대부분 퍼블릭 도메인(출처 표기 권장: "Image: NASA")
가로로 넓은 고해상도(예: 1920x1080 이상) JPG 권장.
