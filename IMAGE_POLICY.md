# 이미지 운영 정책 (헌법 부속 결정 기록)

헌법 제6조는 텍스트 조항이며 이미지 명문 조항은 없다. 이에 다음을 운영 정책으로 박제한다.

1. **오리지널 우선.** 실루엣·키아트·절차적 배경 등 자체 제작 자산을 항상 1순위로 한다.
2. **외부 공식 이미지 채용 조건.** Star Wars Databank 등 공식 출처의 이미지는
   `external_refs` 네임스페이스에 한해 채용하되, 화면에는 반드시
   "이미지 출처: Star Wars Databank © Lucasfilm Ltd." 캡션을 표기한다.
3. **권리자 요청 시 즉시 제거.** 권리자(디즈니/루카스필름)의 요청이 있으면 해당
   이미지를 지체 없이 내린다. 본 아카이브는 비상업 팬 기록물이다.
4. **원문 텍스트는 별개.** 외부 영문 description 원문은 사이트에 노출하지 않는다
   (제6조). cache/ 에만 보관하며 자기 언어 재구성의 소스로만 쓴다.
5. **폴백 의무.** 외부 이미지는 로드 실패(오프라인·URL 변경) 시 자동으로 숨고
   기존 오리지널 레이어로 대체되도록 구현한다.
